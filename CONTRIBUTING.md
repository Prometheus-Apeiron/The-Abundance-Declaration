# Contributing to The Abundance Declaration

This framework is released under CC0 and is designed to be challenged, improved, and expanded. Meaningful contributions of all kinds are welcome.

## Ways to Contribute

### Critique and Feedback
The most valuable contribution is rigorous intellectual critique.
Open a [Discussion](https://github.com/Prometheus-Apeiron/The-Abundance-Declaration/discussions) or [Issue](https://github.com/Prometheus-Apeiron/The-Abundance-Declaration/issues) identifying:
- Empirical claims that are wrong or poorly sourced
- Arguments that don't hold up under scrutiny
- Perspectives and evidence that the framework overlooks
- Sections where the reasoning is weakest
Rule of thumb: Issues are best for bugs/tasks. Discussions are best for open-ended conversation.

### Corrections
If you find factual errors, broken citations, or outdated data, please open an Issue with:
- The specific section and claim
- The correct information with source
- Why the correction matters

### Translations
The Declaration is in English only. Translations into any language are welcome. Please open an Issue to coordinate so efforts aren't duplicated.

### Case Studies
Real-world examples of abundance-oriented institutions, commons
governance, or coordination innovations — especially from the
Global South — would strengthen the framework substantially.

### Building on the Framework
If you develop the ideas further — essays, policy proposals,
local implementations, academic responses — let us know. This
is meant to be a starting point, not a final word.

## How to Submit Changes

1. Fork the repository
2. Create a branch for your changes
3. Make your edits
4. Submit a Pull Request with a clear description of what you
   changed and why

## Code of Conduct

Engage in good faith. Critique ideas, not people. This project
values intellectual rigor, empirical honesty, and constructive
disagreement.
